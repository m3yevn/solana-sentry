Documentation

Solana Sentry is a TypeScript Express service for querying Solana transactions and handling Alchemy webhook notifications with USDC verification.

Install

git clone https://github.com/m3yevn/solana-sentry
cd solana-sentry
npm install
cp .env.example .env   # create from README vars
npm run dev

Requires Node.js 18+.

Environment variables

VariableDescriptionDefault
PORTLocal server port10000
SOLANA_RPC_URLSolana RPC endpointdevnet
SOLANA_ADDRYour wallet address (webhook target)—
USDC_MINTUSDC mint address for transfer checks—

API reference

Health check

GET /health

{ "status": "ok" }

Query transaction

GET /query/:txHash

{
  "txHash": "...",
  "tokens": [{ "mint": "...", "amount": 0.01 }]
}

Webhook

POST /webhook
Content-Type: application/json

# Alchemy transaction webhook payload
# Verifies USDC transfer to SOLANA_ADDR

Webhook setup (Alchemy)

  1. Create an Alchemy app on Solana devnet or mainnet.
  2. Add a webhook pointing to https://your-domain/webhook.
  3. Set SOLANA_ADDR and USDC_MINT in Vercel env.
  4. Send a test USDC transfer — check server logs for verification.

Deploy (Vercel)

vercel --prod

# Set env vars in Vercel dashboard:
# SOLANA_RPC_URL, SOLANA_ADDR, USDC_MINT

Static landing at / and docs at /docs. API routes proxy to the Express server.
